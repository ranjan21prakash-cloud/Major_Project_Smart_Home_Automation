# Major_Project_Smart_Home_Automation

# Project Overview
An IoT-based smart home automation system designed to control lighting and door access using Arduino, ESP32, and IoT technology.

**Created by:** Prakash Ranjan  
**Institution:** National Institute of Technology (NIT), Patna

---

## Features
-  Automated door opening/closing using servo motor
-  Presence detection with ultrasonic sensor
-  Multi-color RGB LED lighting control via Blynk app
-  WiFi-enabled IoT interface
-  Easy-to-use mobile app control

---

## Hardware Components Used

| Component | Purpose |
|-----------|---------|
| ESP32 | WiFi connectivity & Blynk app integration |
| Arduino Uno | Main microcontroller for door control |
| Ultrasonic Sensor | Detect human presence |
| Servo Motor | Automated door opening/closing |
| RGB LED Module | Multi-color lighting control |
| Breadboards & Jumper Wires | Circuit connections |
| Resistors | Current limiting |

---

## Software Requirements
- Arduino IDE (version 1.8+)
- Blynk App (available on Google Play & App Store)
- ESP32 Board Library
- Servo Library

---

## How to Use

1. **Set up the hardware** following the circuit diagram
2. **Upload the Arduino code** to both Arduino Uno and ESP32
3. **Install the Blynk app** on your smartphone
4. **Configure the WiFi credentials** in the code
5. **Control via the Blynk app** interface
