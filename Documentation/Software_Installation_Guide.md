# Software Installation Guide

## Step 1: Install Arduino IDE
1. Go to arduino.cc
2. Download Arduino IDE 1.8.19
3. Install it

## Step 2: Add ESP32 Board
1. Open Arduino IDE
2. Go to Preferences
3. Add: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
4. Install ESP32 boards

## Step 3: Install Libraries
In Arduino IDE, go to Sketch → Include Library → Manage Libraries
- Search and install: Blynk
- Search and install: Servo
- Search and install: DHT sensor library (for future upgrades)

## Step 4: Install Blynk App
- Download from Google Play Store
- Create account
- Create new project
