# How to Set Up Blynk App

## Step 1: Create Blynk Account
1. Download Blynk app
2. Create account with email
3. Verify email

## Step 2: Create New Project
1. Click "New Project"
2. Name: "Smart Home"
3. Device: ESP32
4. Connection: WiFi

## Step 3: Get Auth Token
- Blynk will send you a token via email
- Copy this token (keep it secret!)

## Step 4: Add Widgets
- Click "+" button
- Add "zeRGBa" widget for LED color control

## Step 5: Configure Widgets
- RGB LED: Set to respective GPIO pins

## Step 6: Upload to ESP32
- Paste auth token in your ESP32 code
